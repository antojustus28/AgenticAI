# AgenticAI Platform


## Overview

AgenticAI is a modular .NET solution designed to orchestrate intelligent agents, workflows, and integrations.  

It follows a clean architecture pattern with clear separation of concerns across projects.


## Project Structure

- **AgenticAI.API** → Web API entry point

- **AgenticAI.Core** → Domain models, interfaces, and business logic

- **AgenticAI.Infrastructure** → Persistence, repositories, and external service integrations

- **AgenticAI.Agents** → Specialized agent logic and strategies

- **AgenticAI.Orchestration** → Workflow coordination and pipelines

- **AgenticAI.Tests** → Unit and integration tests


## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/antojustus28/AgenticAI.git
```

2. Create a new repository on the command line

```bash
git config --global user.email "@gmail.com"
git config --global user.name ""

git init
git add README.md
git commit -m "Initial commit - Agentic AI Platform"
git branch -M main
git remote add origin https://github.com/antojustus28/AgenticAI.git
git push -u origin main
```

3. Push an existing repository from the command line

```bash
git remote add origin https://github.com/antojustus28/AgenticAI.git
```
